# Zaplanuj Jedzonko
Simple website created with Django framework and PostgreSQL DB using Scrum methodology.

## Table of contents
* [General info](#general-info)
* [Technologies](#technologies)
* [Requirements](#requirements)
* [Sources](#sources)

## General info
It is a simple website creating for people who wants to plan their meals for better control of taken calories. It provides several features:
* meals CRUD
* create timetables of meals

## Technologies
* Django 2.2.6
* PostgreSQL database

## Requirements
All required packages are mentioned in requirements.txt file. To install it run the following command:
```
pip3 install -r requirements.txt
```
